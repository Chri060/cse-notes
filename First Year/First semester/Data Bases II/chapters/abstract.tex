\begin{abstract}
    The objective of this course is to equip software designers with the skills necessary for proficiently developing database applications.

    Initially, the course delves into the foundational aspects of modern database architectures, highlighting the importance of transactions and their implementation in both centralized and distributed systems.
    
    Subsequently, it explores the advancements in database systems, shedding light on alternative approaches surpassing the traditional relational model. 
    This includes topics such as active databases, object-oriented systems, and XML data management solutions.
\end{abstract}
