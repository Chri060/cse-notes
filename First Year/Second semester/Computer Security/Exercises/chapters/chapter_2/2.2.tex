\section{Exercise two}
Translate the given C code into assembly x86. 
\begin{verbatim}
a = 0;
for(i = 0; i < 10; i++)
    a += i;
\end{verbatim}
Assume $a$ is stored in EAX.

\subsection*{Solution}
The equivalent assembly code for the C program is as follows:
\begin{verbatim}
    mov eax, 0
    mov ebx, 0
    mov ecx, 10
LOOP:
    cmp ebx, ecx
    jge END
    add eax, ebx
    inc ebx
    jmp LOOP
END:
    nop
\end{verbatim}
