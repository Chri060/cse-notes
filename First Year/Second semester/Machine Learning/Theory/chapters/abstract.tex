\begin{abstract}
    The course topics are: 
    \begin{itemize}
        \item Introduction: basic concepts. 
        \item Learning theory: 
            \begin{itemize}
                \item Bias/variance tradeoff. Union and Chernoff/Hoeffding bounds.
                \item VC dimension. Worst case (online) learning.
                \item Practical advice on how to use learning algorithms. 
            \end{itemize}
        \item Supervised learning:
            \begin{itemize}
                \item Supervised learning setup. LMS.\@
                \item Logistic regression. Perceptron. Exponential family. 
                \item Kernel methods: Radial Basis Networks, Gaussian Processes, and Support Vector Machines.
                \item Model selection and feature selection.
                \item Ensemble methods: Bagging, boosting.
                \item Evaluating and debugging learning algorithms.
            \end{itemize}
        \item Reinforcement learning and control: 
            \begin{itemize}
                \item MDPs. Bellman equations. 
                \item Value iteration and policy iteration. 
                \item TD, SARSA, Q-learning.
                \item Value function approximation. 
                \item Policy search. Reinforce. POMDPs. 
                \item Multi-Armed Bandit.
            \end{itemize}
    \end{itemize}
\end{abstract}
