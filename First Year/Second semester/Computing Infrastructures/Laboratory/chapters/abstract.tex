\begin{abstract}
    The course topics are: 
    \begin{itemize}
        \item Hardware infrastructure of datacenters:
            \begin{itemize}
                \item Basic components, rack structure, cooling.
                \item Hard Disk Drive and Solid State Disks.
                \item RAID architectures.
                \item Hardware accelerators.
            \end{itemize}
        \item Software infrastructure of datacenters:
            \begin{itemize}
                \item Virtualization: basic concepts, technologies, hypervisors and containers. 
                \item Computing Architecture: Cloud, Edge and Fog Computing. 
                \item Infrastructure, platform and software-as-a-service.
            \end{itemize}
        \item Methods:
            \begin{itemize}
                \item Scalability and performance of datacenters: definitions, fundamental laws, queuing network theory basics.
                \item Reliability and availability of datacenters: definitions, fundamental laws, reliability block diagrams.
            \end{itemize}
    \end{itemize}
\end{abstract}
